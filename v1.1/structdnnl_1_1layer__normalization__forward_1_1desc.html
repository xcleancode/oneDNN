Initializes a layer normalization descriptor for forward propagation using prop_kind (possible values are dnnl::forward_training and dnnl::forward_inference), memory descriptor data_desc, normalization parameter epsilon, and flags set using bit flags of type dnnl_layer_normalization_desc_t.

Note
In-place operation is supported; that is, dst points to the same memory as src.
