Creates out-of-place concat_primitive_desc for concatenation of n inputs by concat_dimension with resulting output_desc memory descriptor.

output_desc can be NULL or specified with the mkldnn_any format – in this case, the appropriate memory format would be chosen automatically.

Order of inputs:

  • input 0 (mkldnn_query_input_pd, 0)
  • input 1 (mkldnn_query_input_pd, 1)
  • ...
  • input n - 1 (mkldnn_query_input_pd, n - 1)

Order of outputs:

  • output (mkldnn_query_output_pd, 0)
