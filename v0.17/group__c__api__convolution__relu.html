Initializes a merged convolution-relu descriptor conv_relu_desc for forward propagation (supported inference mode only) using convolution descriptor conv_desc and ReLU parameter negative slope.

Deprecated:
use mkldnn_convolution_desc_init with mkldnn_post_ops_append_eltwise to append ReLU

Order of inputs:

  • src (mkldnn_query_src_pd, 0)
  • weights (mkldnn_query_weights_pd, 0)
  • bias (mkldnn_query_weights_pd, 1), if convolution is created with bias

Order of outputs:

  • dst (mkldnn_query_dst_pd, 0)
