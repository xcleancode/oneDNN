dnnl_status_t DNNL_API dnnl_inner_product_forward_desc_init ( dnnl_inner_product_desc_t *  ip_desc,
dnnl_prop_kind_t  prop_kind,
const dnnl_memory_desc_t *  src_desc,
const dnnl_memory_desc_t *  weights_desc,
const dnnl_memory_desc_t *  bias_desc,
const dnnl_memory_desc_t *  dst_desc 
)

Initializes an inner product descriptor ip_desc for forward propagation using prop_kind (possible values are dnnl_forward_training and dnnl_forward_inference) and memory descriptors.

In order to create an inner product without bias, bias_desc should be either NULL or a pointer to a descriptor with memory format kind equals dnnl_format_kind_undef.

Note
Memory descriptors are allowed to be initialized with dnnl_format_kind_any value of format_kind.

Inputs:

  • src (dnnl_query_src_md, 0)
  • weights (dnnl_query_weights_md, 0)
  • bias (dnnl_query_weights_md, 1), if created with bias

Outputs:

  • dst (dnnl_query_dst_md, 0)
